BI450

Data Acquisition and Signal Processing in Biomedical Applications

Introduction

This course is aimed to implement proof of concept of data acquisition (DAQ) and signal processing in biomedical applications (i.e., heart rate measurement and electromyography).

Prerequisite

  • Computer programming
  • Microsystem orEmbedded system
  • Sensor
  • Signal processing

Tools

  • Hardwares (optional)
    • USB-4704
      • DAQNavi
      • DAQNavi Driver for USB-4704
    • Analog Discovery
    • Raspberry Pi 3 Model B
    • TM4C123G LaunchPad
    • Pulse Sensor
    • MyoWare Muscle Sensor
  • Softwares
    • Matlab
    • Qt
    • Keil uVersion
    • etc
